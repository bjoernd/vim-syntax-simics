vim-syntax-simics
=================

VIM syntax highlighting for Simics configuration files

install
=======
To install, issue this command in your linux terminal:

$> curl -s https://raw.github.com/bjoernd/vim-syntax-simics/master/webinstall.sh | /bin/sh

Enjoy!
